Panduan SMC AI-Trading v2.0

Versi 2.0 ini bekerja secara 100% otomatis tanpa perlu mengunggah gambar screenshot lagi.

  1. Input **Gemini API Key** Anda di bar atas, lalu simpan.
  2. Sistem akan mendownload data OHLC lilin (*candles*) real-time dari market untuk 4 TF: **Daily (D1)**, **H4**, **H1**, dan **M15**.
  3. AI kemudian akan meraba struktur market (CHoCH, BOS, FVG, Liquidity sweeps) secara matematis berdasarkan data lilin tersebut.
  4. Jika Anda memilih **Custom**, Anda bebas memasukkan kode aset Binance spot apa saja untuk dianalisa (misal: *SOLUSDT*, *ADAUSDT*).

Catatan Transparansi: Untuk indeks tradisional seperti NAS100 & US30, karena keterbatasan CORS API publik, kami menggunakan pergerakan harga aset crypto terkemuka (BTC) yang diselaraskan dengan kisaran skala indeks tersebut sebagai proksi pola pergerakan pasarnya. Ini memberikan akurasi analisis pola struktur yang tetap realistik.
